## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration

## 功能实现
- Auth：登陆注册改密
- Good：商品信息查询、搜索
    - 商品详情页
    - 商品内容（小）
- Order：商品下单
- Pay：支付
    - 微信支付（WxPay）
